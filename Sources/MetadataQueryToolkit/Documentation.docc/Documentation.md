# ``NSMetadataToolbox``

## Overview

Extensions and conveniences for
[`NSMetadataQuery`](https://developer.apple.com/documentation/foundation/nsmetadataquery)
and
[`NSMetadataItem`](https://developer.apple.com/documentation/foundation/nsmetadataitem)\.
